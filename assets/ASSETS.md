# Asset Notes

This project uses royalty-free placeholder media from Unsplash and Coverr via direct URLs.

## Recommended production assets
- Hero video: licensed clinic interior footage (1080p, compressed with H.264, under 3MB).
- Service images: smiling patients, whitening close-up, implant X-ray visuals.
- Team photos: professional portraits with consistent white/blue background.
- Clinic interior photos: reception, treatment rooms, sterilization zone.

## Optimization guidance
- Export responsive image sizes (480w, 768w, 1200w).
- Use modern formats (`.webp`, `.avif`).
- Keep largest image under 300KB where possible.
